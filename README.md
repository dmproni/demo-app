#
### Запуск

Require Maven & Java 11 on classpath
 
Go to project root.
 
Build

    mvn clean package
    
Launch & Stop
    
    docker-compose up 
    docker-compose down
    
Use

[Web Swagger UI](http://localhost:8080/swagger-ui.html)

[Integration Swagger UI](http://localhost:8081/swagger-ui.html)
